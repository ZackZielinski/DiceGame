Welcome to Guardians and Greeds

This Dungeons and Dragons style of game where you kill enemies in order to get the most amount of loot within a set number of rounds.

NOTE: This does not keep track of the statistics. This is only to provide a set number of dice to this game. This allows any player to set up the board however they like, so you can always play differently.
